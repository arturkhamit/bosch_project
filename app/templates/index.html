Super basic HTML frontend for all existing endpoints. Use the buttons/forms below to call the Flask JSON API.

Response

Click any button to call an endpoint.

Products

POST /product/

Body must be a JSON list.

PATCH /product/<id>

DELETE /product/<id>

Inspectors

POST /inspector/

Body must be a JSON list.

PATCH /inspector/<id>

DELETE /inspector/<id>

Inspections

POST /inspection/

Body must be a JSON list. Create products and inspectors first, then use their IDs.

PATCH /inspection/<id>

DELETE /inspection/<id>

Defects

POST /defect/

Body must be a JSON list. Create a product and inspection first, then use their IDs.

PATCH /defect/<id>

DELETE /defect/<id>

Reports
